# KliqLoanApp
KliqLoanApp Challenge Project

- Used Swift 6 language
- Used MVVM-R design pattern
- Used SPM for library management
- Used tuist for modulating project
- Used SwiftUI for UI desgns
- Used Swift Concurrency
- Used UserDefaults for session persistence

Used third party libraries:

1. Lottie;

For showing loader animation when app fetches data

## Installation

Clone the repo:

```
git clone https://github.com/cagatayegilmez/KliqLoanApp.git
```

Go to the project folder:

```
cd yourFolder/KliqLoanApp
```

Fetch the dependencies:

```
tuist install
```

Generate the project:

```
tuist generate
```

Note: This project uses SwiftLint, so SwiftLint must be installed on your machine to build the project.
